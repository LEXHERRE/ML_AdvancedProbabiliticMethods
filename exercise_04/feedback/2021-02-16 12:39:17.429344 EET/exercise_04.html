def rbf(x, centers, lambdaval):
    # Radial Basis Function output for input x
    #
    # Inputs:
    # x : input points (one-dimensional array)
    # centers : basis function centers (one-dimensional array)
    # lambdaval : basis function width (scalar)
    #
    # Output:
    # Radial Basis Functions evaluated at x (two-dimensional array with len(x)
    #                                        rows and len(centers) columns)
    d = x[:,np.newaxis] - centers[np.newaxis,:]
    y = np.exp(-0.5 * (d ** 2) / lambdaval)
    return y

Problem 3: Poisson regression with Laplace approximation¶

Poisson regression can be used to model count data. A Poisson regression model can be defined as \begin{align} y_i \mid \boldsymbol{\theta} & \sim \operatorname{Poisson}(\exp(\boldsymbol{\theta}^T \mathbf{x}_i )) \\ \theta &\sim \mathcal{N}(\mathbf{0}, \alpha^{-1} \mathbf{I}) \end{align} where $y_i$ are the observed counts, $\mathbf{x}_i$ the related covariates, $i = 1,\ldots,N$, and $\boldsymbol{\theta}^T$ are the regression weights. In this exercise, we approximate the posterior $p(\boldsymbol{\theta} \mid \mathbf{y})$ using the Laplace approximation. We will do this in two steps. In the first step we will derive the gradient $-\nabla \log p(\boldsymbol{\theta} \mid \mathbf{y} )$ and in the second step we will write Laplace approximation. And finally, we will look compare the true density with the laplace approximation.

(a) Derive the gradient $-\nabla \log p(\boldsymbol{\theta} \mid \mathbf{y} )$ and the Hessian $\mathbf{H} = -\nabla\nabla\log p( \boldsymbol{\theta} \mid y)$ needed for the Laplace approximation.

(b) Write the Laplace approximation as the density of a Gaussian distribution. What is the mean and the covariance matrix of this distribution?

(c) Compare the Laplace approximation to the true posterior (computed using numerical integration), in a case where we have one-dimensional covariates only. Use data given in the file 'ex4_4_data.txt' and hyperparameter value $\alpha = 10^{-2}$. Plot the two posteriors and the true value $\theta = \pi/4$ used to generate the data. Also plot the data with the regression line $\hat y_i = \exp(\hat\theta x_i)$ using the MAP estimate $\hat{\boldsymbol{\theta}}$. Solution¶

(a) The negative log posterior is given by \begin{align*} E(\boldsymbol{\theta}) = -\log p(\boldsymbol{\theta}|\mathbf{y}) = - \sum_i \left( y_i \cdot \boldsymbol{\theta}^T\mathbf{x}_i - \exp(\boldsymbol{\theta}^T\mathbf{x}_i) - \log y_i!\right) + \frac{\alpha}{2}\boldsymbol{\theta}^T\boldsymbol{\theta}. \end{align*} Note that in the current context, $E$ stands for energy (a term which derives from physics), not expectation. The gradient with respect to the parameters $\boldsymbol{\theta}$ is then \begin{align*} \frac{\partial E}{\partial \boldsymbol{\theta}} = - \sum_i\left(y_i\mathbf{x}_i - \mathbf{x}_i\exp(\boldsymbol{\theta}^T\mathbf{x}_i)\right) + \alpha\boldsymbol{\theta} \end{align*} and the Hessian \begin{align*} \mathbf{H} = \frac{\partial^2 E}{\partial\boldsymbol{\theta}\partial\boldsymbol{\theta}^T} = + \sum_i \mathbf{x}_i\mathbf{x}_i^T\exp(\boldsymbol{\theta}^T\mathbf{x}_i) + \alpha\mathbf{I}. \end{align*} (b) The Laplace approximation is a Gaussian density $\mathcal{N}(\boldsymbol{\theta}|\boldsymbol{\theta}^*,\mathbf{S})$, where the mean $\boldsymbol{\theta}^*$ is the value of the parameter for which the gradient of the log posterior is 0, and the covariance matrix $\mathbf{S} = \mathbf{H}^{-1}|_{ \boldsymbol{\theta}^*}$ is the inverse of the Hessian evaluated at $\boldsymbol{\theta}^*$.

(a)

First of all we know that:

$$p(\mathbf{y}|\boldsymbol{\theta},\alpha) = \prod_{i=1}^n \frac{\exp\left(-\exp\left(\boldsymbol{\theta}^\top \mathbf{x}_i\right)\right) \times \exp\left(y_i \boldsymbol{\theta}^\top \mathbf{x}_i\right)}{y_i!}$$$$p(\boldsymbol{\theta}|\alpha) = \left(\frac{\alpha}{2\pi}^\frac{\beta}{2}\right) \times \exp\left(-\frac{\alpha}{2}\boldsymbol{\theta}^\top\boldsymbol{\theta}\right)$$$$p\left(\boldsymbol{\theta}|\mathbf{y},\alpha\right) = \frac{p\left(\mathbf{y}|\boldsymbol{\theta},\alpha\right) p\left(\boldsymbol{\theta}|\alpha\right)}{p\left(\mathbf{y}|\alpha\right)} \propto p\left(\mathbf{y}|\boldsymbol{\theta},\alpha\right) p\left(\boldsymbol{\theta}|\alpha\right)$$

So we can obtain $\log{p(\boldsymbol{\theta}|\mathbf{y},\alpha)}$ as:

$$\log{p\left(\boldsymbol{\theta}|\mathbf{y},\alpha\right)} \propto \log{\left(p\left(\mathbf{y}|\boldsymbol{\theta},\alpha\right) p\left(\boldsymbol{\theta}|\alpha\right)\right)} = \log{p\left(\mathbf{y}|\boldsymbol{\theta},\alpha\right)} + \log{p\left(\boldsymbol{\theta}|\alpha\right)} =$$$$= \log{\left(\prod_{i=1}^n \frac{\exp\left(-\exp\left(\boldsymbol{\theta}^\top \mathbf{x}_i\right)\right) \times \exp\left(y_i \boldsymbol{\theta}^\top \mathbf{x}_i\right)}{y_i!}\right)} + \log{\left(\left(\frac{\alpha}{2\pi}^\frac{\beta}{2}\right) \times \exp\left(-\frac{\alpha}{2}\boldsymbol{\theta}^\top\boldsymbol{\theta}\right)\right)} \propto$$$$\propto \sum_{i=1}^n\left(-\exp\left(\boldsymbol{\theta}^\top \mathbf{x}_i\right)+y_i\boldsymbol{\theta}^\top \mathbf{x}_i\right) - \frac{\alpha}{2}\boldsymbol{\theta}^\top\boldsymbol{\theta}$$

Now we can derive the negative gradient as:

$$-\nabla\log{p(\boldsymbol{\theta}|\mathbf{y})} = - \frac{\delta\left(\sum_{i=1}^n\left(-\exp\left(\boldsymbol{\theta}^\top \mathbf{x}_i\right)+y_i\boldsymbol{\theta}^\top \mathbf{x}_i\right) - \frac{\alpha}{2}\boldsymbol{\theta}^\top\boldsymbol{\theta}\right)}{\delta\boldsymbol{\theta}} =$$$$= \sum_{i=1}^n\left(\exp\left(\boldsymbol{\theta}^\top\mathbf{x}_i\right)\mathbf{x}_i^\top - y_i\mathbf{x}_i^\top\right) + \alpha\boldsymbol{\theta}^\top$$

And the Hessian is:

$$\mathbf{H} = -\nabla\nabla \log{p\left(\boldsymbol{\theta}|\mathbf{y}\right)} = - \frac{\delta\left(\sum_{i=1}^n\left(-\exp\left(\boldsymbol{\theta}^\top\mathbf{x}_i\right)\mathbf{x}_i^\top + y_i\mathbf{x}_i^\top\right) - \alpha\boldsymbol{\theta}^\top\right)}{\delta\boldsymbol{\theta}} = \sum_{i=1}^n\left(\mathbf{x}_i\exp\left(\boldsymbol{\theta}^\top\mathbf{x}_i\right)\mathbf{x}_i^\top\right) + \alpha\mathbf{I}$$

Now knowing that the Laplace approximation is:

$$\tilde{E}(\boldsymbol{\theta}) = E(\boldsymbol{\bar{\theta}}) + \frac{1}{2}(\boldsymbol{\theta} - \bar{\boldsymbol{\theta}})^\top \mathbf{H}_{\boldsymbol{\theta}} (\boldsymbol{\theta} - \bar{\boldsymbol{\theta}})$$

Where $E(\boldsymbol{\theta}) = - \log{p(\boldsymbol{\theta}|\mathbf{y},\alpha)} \propto \sum_{i=1}^n\left(\exp\left(\boldsymbol{\theta}^\top\mathbf{x}_i\right) - y_i\boldsymbol{\theta}^\top\mathbf{x}_i\right) + \frac{\alpha}{2}\boldsymbol{\theta}^\top\boldsymbol{\theta}$

We can observe that the function follows a Gaussian approximation $q(\boldsymbol{\theta}|\mathbf{y},\alpha) \propto \exp\left(-\tilde{E}(\boldsymbol{\theta})\right)$ as:

$$q(\boldsymbol{\theta}|\mathbf{y},\alpha) \sim \mathcal{N}(\boldsymbol{\theta}|\mathbf{m},\mathbf{S})$$

where, $\mathbf{S} = \mathbf{H}_{\boldsymbol{\theta}}^{-1}$ and $\mathbf{m} = \boldsymbol{\bar{\theta}}$ and where $\mathbf{H}_{\boldsymbol{\theta}}^{-1}$ is the Hessian $\mathbf{H}_{\boldsymbol{\theta}}$ of $E(\boldsymbol{\theta})$ at $\bar{\boldsymbol{\theta}}$ and $\boldsymbol{\bar{\theta}}$ is the minimum value of $E(\boldsymbol{\theta})$ that can be found analytically or by numerical optimization. .
